# 印刷コース（A4タイル方式）

ライントレース用の **白地に黒線** コースを A4 で印刷して組む。
ファームの `LINE_IS_DARK = true`（明るい床に黒線）に合わせてある。

## ファイル

- `print.html` … 印刷用。1ページ目=説明＋原寸チェック定規、2ページ目=直線タイル、3ページ目=90°カーブタイル

## 印刷手順

1. `print.html` をブラウザ（Chrome / Safari）で開く。
2. 印刷 → **倍率 100%**、**「用紙に合わせる」OFF**、余白なし、用紙 **A4縦**。
   - Chrome: 詳細設定 → 倍率「カスタム 100」、余白「なし」、背景のグラフィック ON
3. 1 枚目を印刷して定規が **100mm ちょうど**か確認（原寸確認）。ずれたら倍率を直す。
4. 必要枚数を印刷：
   - **最小の周回コース** = カーブ 3 枚目を **4 枚** → 2×2 で円形の輪
   - 直線を足して長くするなら直線 2 枚目を必要数

## 組み立て

- 各タイルの**外周の破線（180mm 角）で切る**。四隅のトンボが目印。
- 切った辺をぴったり突き合わせ、**裏からテープ**で留める（表に黒テープを出さない＝誤検出防止）。
- タイルは正方形なので**紙を回して**向きを合わせる（直線は縦横自在、カーブは 4 方向）。
- 床に平らに敷く。浮き・段差・影は黒線と誤検出されるので避ける。

## 仕様（タイル）

| 項目 | 値 |
|------|----|
| タイル寸法 | 180mm 角（切り取り線） |
| 線色 / 線幅 | 黒 / 30mm |
| 線の通り道 | 各辺の中央（隣接タイルと必ずつながる） |
| カーブ半径 | 90mm（中心線）。やや急なので低速推奨 |

線幅・半径を変えたいときは `print.html` の `stroke-width="30"` と
カーブの `A 90 90 ...`（半径 90）を編集する。

## 走行とチューニング

1. シリアル(115200) `i`（IDLE）で `det=Y` と `cx/err` がコース上で追従するか確認
2. `l` で走行。`s` で停止
3. うまくいかない時の目安（ファーム冒頭の定数）
   - 線を拾わない / `det=N` が多い → `LINE_MARGIN` を下げる
   - 影や外光を線と誤検出する → `LINE_MARGIN` を上げる、`LOCAL_BG_R` を広げる（局所平均の横幅を大きく）
   - カーブで外れる → `CORNER_MAX_SPEED` を下げる（コーナー速度上限）、`CORNER_SLOWDOWN` を上げる、`KP` を上げる
   - 高速(240BPM等)で曲がりきれない → まず `CORNER_MAX_SPEED` を下げて確実に減速させる
   - 直線でフラつく → `KP` を下げる
